# Kanban Advanced | Vitty Dev

Uma ferramenta de produtividade robusta e moderna desenvolvida com foco nos princípios de **Engenharia de Software** e **Metodologias Ágeis**. Este sistema combina a visualização clássica do quadro Kanban com métricas de desempenho em tempo real e gestão de tempo avançada.

---

## Funcionalidades Principais

### Quadro Kanban Interativo
- **Drag and Drop:** Movimentação fluida de tarefas entre as colunas (A Fazer, Fazendo e Feito).
- **Gestão de Prazos:** Inclusão de data e hora em cada tarefa para controle de cronograma.
- **Feedback Auditivo:** Efeito sonoro exclusivo ao concluir uma tarefa (coluna Feito).
- **Acessibilidade:** Atalho via teclado (tecla Enter) para criação rápida de novos itens.

### Dashboard de Métricas
- **Gráfico Dinâmico:** Gráfico de rosca (*Doughnut*) utilizando a biblioteca **Chart.js**.
- **Indicadores de Progresso:** Exibição da porcentagem total de conclusão centralizada no dashboard.
- **Contadores em Tempo Real:** Atualização imediata do número de tarefas em cada estágio do fluxo.

### Cronômetro de Alta Precisão
- **Contagem Regressiva:** Suporte para Dias, Horas, Minutos e Segundos.
- **Controle Total:** Funções de Iniciar, Pausar (com retomada inteligente) e Resetar.
- **Alarme:** Aviso sonoro e visual ao término da contagem.

---

## Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização avançada com Flexbox, Grid e animações
- **JavaScript (Vanilla):** Lógica de manipulação de DOM, Drag and Drop API e processamento do cronômetro.
- **Chart.js:** Biblioteca externa para renderização de gráficos estatísticos.
- **FontAwesome:** Ícones para melhoria da experiência do usuário (UX).
